cvt.models.KernelMSM¶

class cvt.models.KernelMSM(n_subdims, normalize=False, sigma=None, faster_mode=False)¶

Kernel Mutual Subspace Method

__init__(n_subdims, normalize=False, sigma=None, faster_mode=False)¶
Parameters

  • n_subdims (int) – The dimension of subspace. it must be smaller than the dimension of original space.

  • normalize (boolean, optional (default=True)) – If this is True, all vectors are normalized as |v| = 1

  • sigma (int or str, optional (default=None)) – a parameter of rbf kernel. if sigma is None, sqrt(n_dims / 2) will be used.
